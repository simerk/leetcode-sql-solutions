## 🚀 586. Customer Placing the Largest Number of Orders

### 💡 Idea

The goal is to find the customer who has placed the **highest number of orders**.

First, group the orders by `customer_number` and count the number of orders placed by each customer. Then, sort the counts in descending order and return the customer with the highest count.

---

### 💻 Solution

```sql
SELECT customer_number
FROM (
    SELECT customer_number, COUNT(*) AS order_count
    FROM Orders
    GROUP BY customer_number
    ORDER BY order_count DESC
    LIMIT 1
) AS t;
```

---

### 🛠️ How It Works

1. Group all orders by `customer_number`.
2. Count the number of orders placed by each customer using `COUNT(*)`.
3. Sort the customers by their order count in descending order.
4. Use `LIMIT 1` to select the customer with the highest number of orders.
5. Return the `customer_number`.

---

### ⏱️ Complexity

**Time Complexity:** `O(n log n)` *(grouping followed by sorting)*

**Space Complexity:** `O(n)` *(stores one grouped record per customer)*
