## 🚀 620. Not Boring Movies

### 💡 Idea

The goal is to find all movies that satisfy the following conditions:

- The movie has an **odd-numbered `id`**.
- The movie's `description` is **not** `'boring'`.

Finally, sort the resulting movies by their `rating` in **descending order**.

---

### 💻 Solution

```sql
SELECT * FROM Cinema
WHERE MOD(id, 2) != 0
AND description != 'boring'
ORDER BY rating DESC;
```

---

### 🛠️ How It Works

1. Select all columns from the `Cinema` table.
2. Use `MOD(id, 2) != 0` to keep only movies with odd IDs.
3. Exclude movies whose `description` is `'boring'`.
4. Sort the remaining movies by `rating` in descending order.

---

### ⏱️ Complexity

**Time Complexity:** `O(n log n)` *(linear scan followed by sorting on `rating`)*

**Space Complexity:** `O(1)` *(excluding the output result set)*
