## 🚀 1407. Top Travellers

### 💡 Idea

The goal is to calculate the total distance traveled by each user.

Since some users may not have any rides, we use a **LEFT JOIN** to include all users. We then sum the distances of their rides and replace `NULL` values with `0` for users who haven't traveled.

Finally, the results are sorted by:
- Total traveled distance in **descending** order.
- User name in **ascending** order to break ties.

---

## 💻 Solution

```sql
SELECT u.name, IFNULL(SUM(r.distance), 0) AS travelled_distance
FROM Users u
LEFT JOIN Rides r
ON u.id = r.user_id
GROUP BY u.id, u.name
ORDER BY travelled_distance DESC, u.name;
```

---

### 🛠️ How It Works

1. Start with the `Users` table so every user is included.
2. `LEFT JOIN` the `Rides` table using `user_id`.
3. Sum the `distance` of all rides for each user.
4. Use `IFNULL()` to replace `NULL` with `0` for users who have no rides.
5. Group the results by user.
6. Sort by:
   - Total traveled distance (descending).
   - User name (ascending).

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single join followed by grouping)*

**Space Complexity:** `O(n)` *(stores grouped users)*
