## 🚀 1731. The Number of Employees Which Report to Each Employee

### 💡 Idea

The goal is to find, for each manager:

- The manager's employee ID.
- The manager's name.
- The number of employees who directly report to them.
- The average age of those employees (rounded to the nearest integer).

Since both managers and employees are stored in the same table, we use a **self join** to match each employee with their manager.

---

## 💻 Solution

```sql
SELECT m.employee_id, m.name,
COUNT(e.employee_id) AS reports_count,
ROUND(AVG(e.age), 0) AS average_age
FROM Employees m
JOIN Employees e
ON m.employee_id = e.reports_to
GROUP BY m.employee_id
ORDER BY m.employee_id;
```

---

### 🛠️ How It Works

1. Treat the `Employees` table as two separate tables:
   - `m` → managers.
   - `e` → employees.
2. Join them where:
   - `m.employee_id = e.reports_to`.
3. Group the records by each manager.
4. Count the number of employees reporting to each manager using `COUNT()`.
5. Calculate the average age of the reporting employees using `AVG()`.
6. Round the average age to the nearest integer.
7. Sort the results by `employee_id`.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single self join followed by grouping)*

**Space Complexity:** `O(n)` *(stores grouped managers)*
