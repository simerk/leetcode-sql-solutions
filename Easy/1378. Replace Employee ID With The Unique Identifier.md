## 🚀 1378. Replace Employee ID With The Unique Identifier

### 💡 Idea

The goal is to replace each employee's `id` with their corresponding `unique_id`.

Since not every employee has a unique identifier assigned, we use a **LEFT JOIN** to ensure all employees appear in the result. Employees without a matching `unique_id` will have `NULL` in the output.

---

## 💻 Solution

```sql
SELECT eu.unique_id, e.name
FROM Employees e
LEFT JOIN EmployeeUNI eu
ON e.id = eu.id;
```

---

### 🛠️ How It Works

1. Start with the `Employees` table so that every employee is included.
2. Perform a `LEFT JOIN` with the `EmployeeUNI` table using the `id` column.
3. Retrieve the corresponding `unique_id` for each employee.
4. If an employee has no matching record in `EmployeeUNI`, the `unique_id` will be `NULL`.
5. Return the `unique_id` and employee `name`.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single join)*

**Space Complexity:** `O(1)` *(excluding the output result)*
