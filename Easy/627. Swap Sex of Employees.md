## 🚀 627. Swap Sex of Employees

### 💡 Idea

The goal is to **swap the values of `'m'` and `'f'`** in the `sex` column without using a temporary table or multiple update statements.

We can use a `CASE` expression inside a single `UPDATE` statement to replace:
- `'m'` with `'f'`
- `'f'` with `'m'`

---

### 💻 Solution

```sql
UPDATE Salary
SET sex = CASE
    WHEN sex = 'm' THEN 'f'
    ELSE 'm'
END;
```

---

### 🛠️ How It Works

1. Update every row in the `Salary` table.
2. Use a `CASE` expression to check the current value of `sex`.
3. If the value is `'m'`, change it to `'f'`.
4. Otherwise, change `'f'` to `'m'`.
5. The update is performed in a single SQL statement.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(updates each row once)*

**Space Complexity:** `O(1)` *(no additional storage required)*
