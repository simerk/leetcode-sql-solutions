## 🚀 3451. Find Invalid IP Addresses

### 💡 Idea

The goal is to find all **invalid IPv4 addresses** and count how many times each invalid address appears.

According to the problem, an IP address is invalid if:

- It contains **less or more than 4 octets**.
- Any octet is **greater than 255**.
- Any octet contains **leading zeros** (except the single digit `0`).

We use string functions to extract each octet, validate it, and then count the occurrences of every invalid IP address.

---

## 💻 Solution

```sql
SELECT ip, COUNT(*) AS invalid_count
FROM logs
WHERE
    -- Less or more than 4 octets
    LENGTH(ip) - LENGTH(REPLACE(ip,'.','')) <> 3
    OR
    -- Check if any octet is greater than 255
    CAST(SUBSTRING_INDEX(ip,'.',1) AS UNSIGNED) > 255
    OR CAST(SUBSTRING_INDEX(SUBSTRING_INDEX(ip,'.',2),'.',-1) AS UNSIGNED) > 255
    OR CAST(SUBSTRING_INDEX(SUBSTRING_INDEX(ip,'.',3),'.',-1) AS UNSIGNED) > 255
    OR CAST(SUBSTRING_INDEX(ip,'.',-1) AS UNSIGNED) > 255
    OR
    -- Check for leading zeros
    SUBSTRING_INDEX(ip,'.',1) REGEXP '^0[0-9]+$'
    OR SUBSTRING_INDEX(SUBSTRING_INDEX(ip,'.',2),'.',-1) REGEXP '^0[0-9]+$'
    OR SUBSTRING_INDEX(SUBSTRING_INDEX(ip,'.',3),'.',-1) REGEXP '^0[0-9]+$'
    OR SUBSTRING_INDEX(ip,'.',-1) REGEXP '^0[0-9]+$'
GROUP BY ip
ORDER BY invalid_count DESC, ip DESC;
```

---

### 🛠️ How It Works

1. Count the number of dots using `LENGTH()` and `REPLACE()` to ensure the IP contains exactly **4 octets**.
2. Extract each octet using `SUBSTRING_INDEX()`.
3. Convert each octet to a number using `CAST()` and check if it is greater than **255**.
4. Use `REGEXP` to detect octets with **leading zeros**.
5. Count the occurrences of each invalid IP address and sort the result by `invalid_count` and `ip` in descending order.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)`

**Space Complexity:** `O(n)` *(for the grouped result)*
