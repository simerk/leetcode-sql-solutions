## 🚀 185. Department Top Three Salaries

### 💡 Idea

The goal is to find the employees who have the **top three distinct salaries** in each department.

We can use the `DENSE_RANK()` window function to rank employees by salary **within each department**. Employees with the same salary receive the same rank, ensuring that ties are handled correctly. Finally, we return only those employees whose rank is **3 or less**.

---

### 💻 Solution

```sql
select department, employee, salary
from (
  select d.name as department, e.name as employee, e.salary as salary,
  dense_rank() over (partition by e.departmentId order by e.salary desc) as r 
  from Employee e
  inner join Department d
  on e.departmentId = d.id) as t
WHERE r <= 3;
```

---

### 🛠️ How It Works

1. Join the `Employee` and `Department` tables using `departmentId`.
2. Partition the data by `departmentId` so each department is ranked independently.
3. Sort salaries within each department in descending order.
4. Use `DENSE_RANK()` to assign ranks based on salary.
5. Employees with the same salary receive the same rank.
6. Filter the results to keep only employees with a rank of **3 or less**.
7. Return the department name, employee name, and salary.

---

### ⏱️ Complexity

**Time Complexity:** `O(n log n)` *(sorting salaries within each department for ranking)*

**Space Complexity:** `O(n)` *(stores ranking information for each employee)*
